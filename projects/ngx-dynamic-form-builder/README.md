# 🚀 NGX Dynamic Form Builder

[![GitHub Package](https://img.shields.io/badge/GitHub-Package-blue?logo=github)](https://github.com/MD-MEHBUB-UL-ISLAM/ngx-dynamic-form-builder/packages)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Angular](https://img.shields.io/badge/Angular-%3E%3D12.0.0-red)](https://angular.io/)

JSON-driven dynamic form builder for Angular with 15+ field types, validation, and conditional logic.

## ✨ Features

- 📝 **15+ Field Types**: text, email, password, number, textarea, select, multiselect, radio, checkbox, switch, date, time, color, range, file
- ✅ **Comprehensive Validation**: required, email, minLength, maxLength, pattern, custom sync/async validators
- 🔗 **Conditional Logic**: show/hide fields based on form values
- 🎨 **Flexible Layouts**: vertical, horizontal, inline
- 📦 **Nested Forms**: groups and arrays support
- 🎯 **Custom Templates**: override field rendering
- 🔄 **Reactive Forms**: full integration with Angular's reactive forms
- 📱 **Responsive**: works on all screen sizes
